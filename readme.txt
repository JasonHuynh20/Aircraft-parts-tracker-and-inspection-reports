Files:

Control.h - header file that defines the members and functions for the a control structure to run the program

Control.cc - source file which implements the Control structure



Aircraft.h - header file that defines the members and functions for the Aircraft object

Aircraft.cc - source file that implements the functions for a Aircraft object



Airline.h - header file that defines the members and functions for the Airline object

Airline.cc - source file that implments the functions for the Airline object



Array.h - header and source file that defines and implements the members and functions for the template

	Array objects



Date.h - the header file that defines the members and functions for a Date object

Date.cc - the source file that implements the functions for Date object



Part.h - the header file that defines the members and functions for the abstract and derived part, FH_part,

	IT_Part, and FHIT_part objects

Part.cc - the source file that implements the functions for all Part objects



View.h - the header file that defines the members and functions for displaying the menu

View.cc - the source file that implements the functions for displaying the menu



main.cc - a source file that contains functions and the main function for testing the files above



defs.h - a header file that defines constant variables



Makefile - a make file that compiles and links all nessescary files





Compilation Instructions: In the commandline, type "make" then click enter. the make file will compile and link

all nessecary files.



Execution Instructions: After compilations and linking, type "./a4" then click enter. this will run the

executable file generated by the Make file. Follow the program by typing in the options to execute, then

and other information it requests.
